What is programming language?

A programming language is a way for programmers to communicate with computers. Programming languages consist of a set of rules that allows values to be converted into various ways of generating machine code.

×

why do we need a programming language?

Imagine you want to communicate with a computer to make it do something specific, like playing a game or solving a math problem. Computers only understand a language called "Machine Code," which is a set of binary numbers (0s and 1s). However, writing instructions in machine code is incredibly complex and challenging for humans.So, to make things easier for us, we use programming languages.
Think of a programming language like a translator between humans and computers. It allows us to write instructions in a way that's much more understandable for us, and then it translates those instructions into machine code that the computer can execute.


Example: Making a Sandwich


Let's use the analogy of making a sandwich to understand why we need a programming language:


  1. Machine Code (Binary): Imagine if making a sandwich was represented in machine code – a series of 0s and 1s. It might look something like this: 100110110101101010... This is impossible for us to understand and remember for simple tasks like making a sandwich.


  2. Programming Language (English): Now, think of a programming language as a step-by-step recipe in English. It could be something like:


                                
                     takeBread();
                     spreadPeanutButter();
                     addJelly();
                     putSlicesTogether();

  3. Translation to Machine Code: A computer, with the help of a special program called a compiler or interpreter, takes our English-like instructions and translates them into the complex machine code it understands. The computer understands our sandwich-making instructions and follows them precisely, just like a chef following a recipe.

What is Java?

Java is a versatile programming language that enables developers to create a wide range of software applications. It's known for its simplicity, readability, and platform independence.


  1. Simple: Java is a simple programming language and easy to understand because it does not contain complexities that exist in prior programming languages.


  2. Object-Oriented: Java is an Object Oriented Programming Language, which means in Java everything is written in terms of classes and objects.


  3. Platform independent: The design objective of java people is to develop a language that must work on any platform. Here platform means a type of operating system and hardware technology. Java allows programmers to write their program on any machine with any configuration and to execute it on any other machine having different configurations.


  4. Portable: The WORA (Write Once Run Anywhere) concept and platform-independent feature make Java portable.


  5. Robust: The Java Programming language is robust, which means it is capable of handling unexpected termination of a program.


  6. Memory Management: Java features automatic memory management through garbage collection. Developers don't have to explicitly manage memory allocation and deallocation, reducing the chances of memory-related errors.


  7. Secure: Java also provides security features to the programmers. Security problems like virus threats, tampering, eavesdropping, and impersonation can be handled or minimized using Java.


  8. Multi-threading: In java it enables to create a program that executes multiple tasks concurrently.

History of Java

James Gosling initiated Java language project in June 1991 for use in one of his many set-top box projects. The language, initially called 'Oak' after an oak tree that stood outside Gosling's office, also went by the name 'Green' and ended up later being renamed as Java, from a list of random words.

1. Java 1.0 (1995):
Description: Think of this as Java's "birth." It was created by James Gosling and Mike Sheridan at Sun Microsystems.
Features: Like the first version of a video game, Java 1.0 allowed developers to write and run simple programs.
project: Initially named "Oak" and aimed at creating software for consumer electronics.

2. Java 2 (J2SE 1.2, 1998):
Description: A big upgrade! It's like getting new levels and features in a game.
Features: Significant enhancements like Swing GUI toolkit, Collections Framework, and the "assert" keyword.

3. Java 2EE (Early 2000s):
Description: Imagine Java leveling up to handle bigger tasks, like building powerful websites and business applications. Introduction of J2EE (Java 2 Platform, Enterprise Edition) for enterprise-level applications.
Features: Emergence of popular frameworks like Spring and Hibernate.

4. Java 17 (2023):
Description: Java keeps getting better and stronger. Version 17 is like having the latest version of your favorite game.
Features: Thriving ecosystem with a vast array of libraries, frameworks, and tools.

First Java Program

Introduction:
Welcome to the exciting world of programming! We'll explore a simple Java program that is often the first program people write when learning a new programming language. It's called the "Hello World" program because its main purpose is to display the words "Hello, java!" on the screen. 


                            
                     // This is a simple Java program that prints "Hello, World!" to the console.
                        public class HelloWorld {
                            public static void main(String[] args) {
                                // The next line prints the text "Hello, World!" to the console.
                                System.out.println("Hello, Java!");
                            }
                        }

Breaking it Down:
Let's understand the different parts of this program:

public class HelloWorld { ... }: This is the starting point of our program. It declares a class named HelloWorld, which is a container for our code.


public static void main(String[] args) { ... }: Inside the HelloWorld class, we have a method (a block of code) called main. This is the entry point of our program. It's where the computer starts executing our code.


System.out.println("Hello, Java!");: This line is a command that tells the computer to print something on the screen. In this case, it's printing the text "Hello, java!".
