Increment & Decrement operators in Java

The Java programming language is highly regarded for its readability and simplicity. The usage of numerous operators is one of the elements that contribute to this simplicity. The increment (++) and decrement (-) operators, for example, are used to increase or reduce the value of a variable by one.
In Java, the increment (++) and decrement (--) operators are unary operators, which means they only act on one operand. They are used to add or subtract one from the value of an integer, floating-point, or character variable. There are two methods to use these operators: prefix and postfix.

Increment Operator:
The increment operator (++) adds 1 to the current value of the variable.
Syntax:

variable++; // Postfix increment
++variable; // Prefix increment
